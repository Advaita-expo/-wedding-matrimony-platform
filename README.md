## ğŸš€ How to Use

### 1. Open in Browser
Simply open `index.html` in any modern web browser

### 2. File Structure
Keep all files in the same directory:
- HTML files in root
- styles.css in root
- script.js in root
- All linked images use external URLs (Unsplash)

### 3. Customization

**Update Company Information:**
- Contact details in footer and contact page
- Company name and branding
- Business hours and address

**Change Colors:**
In `styles.css`, modify the `:root` variables:
```css
:root {
    --primary-color: #c41e3a;      /* Main brand color */
    --secondary-color: #f4d03f;    /* Accent color */
    --dark-color: #1a1a1a;         /* Dark text */
    --light-color: #f8f9fa;        /* Light backgrounds */
}
```

**Add Your Images:**
Replace image URLs in HTML files with your own wedding photos

**Update Pricing:**
Modify package prices and features in `pricing.html`
